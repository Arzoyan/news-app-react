# News List APP

## Description

Using this project you can Adding, Editing , deleting and Searching of News witch you created

## Installation

```bash
$ git clone https://github.com/Arzoyan/news-app-react.git
```

navigate to root directory

```bash
$ cd news-app-react

```

Install project

```bash
$ npm install
```

## Running the app

```bash
$ npm run dev
```

## Running the server

```bash
$ npm run start:json-server
```

## Technology

- React
- Typescript
- Redux Toolkit
- Redux-Saga
- Material-UI
- Formik
- React-Router-Dom
- Json-Server
